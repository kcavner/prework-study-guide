HTML

  • the organization of HTML can be done by tags
  • head information contains metadata that is required by the browser to render the page correctly
  • elements included in the head tag are not visible to users when the page is visited.
  • body contains all content we want shown to the user. All elements we want visible and interactable with the user must be included in body
  • !DOCTYPE is not case sensistive because it is a declaration
  • the title of the webpage important to search engine optimization (more frequent search results)

CSS

  • style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS stlyes elements directly in the HTML file
  • Inline CSS is useful for quick and permanent changes however they are not as flexible as external because the editing is less automated
  • Internal CSS is more automated because it is located in the head section of the HTML file and can apply stlye to different classes and ids
  • external CSS is the most popular way of styling a page because it is more organized than having more bulk in the HTML document
  • linking an external CSS file is done by including a link element placed in the HTML's head section
  • external css files is related to the separation of concerns it is good design practice

Javascript

  • functions are a set of instructions that tells the computer how to perform tasks
  • variables are declared by the var keyword
  • assignment operator is =
  • Console is a test environment in browsers that devs use to check their code
  • method is a set of instructions that can be executed by a computer
  • linking javascript files is done by adding between and
  • CSS link is with element. Javascript link is with element
  • console.log method was created for debuging and testing purposes
  • If statements are considered conditional logic and are a part of decision-making
  • Strict equality operator === checks 2 values to be equal and returns boolean results true and false
  • arrays are stored groups of related data
  • arrays are stored in brackets [] with items separated by commas
  • indexes are unique places inside an array with an associated number
  • arrays have their first index as 0 not 1
  • Loops are useful for logging large numbers of indexes
  • Array loops are commonly used for automating repetitive tasks
